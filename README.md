# Gerenciamento de Obras Públicas

Esta API foi criada para gerenciar diversos elementos que compôem uma obra pública. 

Entidades gerenciadas:
- Diários de Obra
- Endereços
- Equipamentos
- Etapas da Obra
- Fiscalizações
- Fornecedores
- Materiais
- Obras
- Relatórios
- Responsáveis Técnicos
